E) Quelques notions à travers des images

La structure d'une page

Pour qu'une page HTML soit déclarée valide, elle doit obligatoirement comporter certains éléments et suivre un schéma précis. Une page non valide sera mal comprise par le navigateur et risque d'impacter négativement le référencement du site dans les moteurs de recherche.

Création d'un formulaire

Les formulaires HTML permettent d'envoyer des données à manipuler ou stocker : inscriptions, connexions, messages, commentaires… Ils peuvent contenir des champs texte, listes d'options, cases à cocher, etc.

Les tableaux

Les tableaux HTML permettent de présenter des données de manière organisée afin de les structurer et de les rendre compréhensibles pour les navigateurs, moteurs de recherche et utilisateurs.

Les CSS

Les CSS (Cascading StyleSheets) sont utilisées pour définir l'apparence du contenu HTML : polices, couleurs, tailles, espacement, colonnes et animations. Elles complètent le HTML qui, lui, gère la structure et la sémantique.
