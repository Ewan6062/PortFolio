Titre : Certification JavaScript — Formation OpenClassRooms

Elle a été réalisée via la plateforme OpenClassRooms et avait pour objectif de développer mes compétences en programmation JavaScript, notamment en manipulation du DOM, en gestion des événements et en dynamisation des interfaces web côté client.

E) Quelques notions à travers des images

Les variables et types de données

En JavaScript, les variables permettent de stocker des informations (texte, nombres, booléens). Elles sont essentielles pour manipuler des données et créer des interactions dynamiques dans une page web.

Les conditions

Les structures conditionnelles (if, else) permettent d'exécuter du code en fonction de certaines situations. Elles sont utilisées pour créer des comportements dynamiques selon les actions de l'utilisateur.

Les fonctions

Les fonctions permettent de regrouper du code réutilisable. Elles facilitent l'organisation du programme et permettent d'exécuter des actions spécifiques à la demande.

Manipulation du DOM

Le DOM permet à JavaScript d'interagir avec les éléments HTML : modifier du texte, ajouter des éléments ou réagir aux actions de l'utilisateur comme les clics.

Les événements

Les événements permettent de détecter les actions de l'utilisateur (clic, saisie clavier, survol) et d'y répondre en temps réel pour rendre le site interactif.
