Titre : Certification Bootstrap — Formation OpenClassRooms

A) Le contexte

Cette certification s'inscrit dans le cadre de ma formation BTS SIO. Elle a été réalisée via la plateforme OpenClassRooms et avait pour but de consolider mes compétences en création de sites web responsives grâce au framework Bootstrap.

Lien de la formation : OpenClassRooms — Bootstrap

B) Mes objectifs

Utiliser le framework Bootstrap
Comprendre le système de grille (grid system)
Créer des pages web responsives facilement
Utiliser les classes utilitaires de Bootstrap
Intégrer des composants (navbar, cards, boutons)
Personnaliser l'apparence avec Bootstrap
Optimiser l'affichage sur mobile et tablette

C) Pourquoi cette formation ?

Approfondir mes compétences en développement front-end
Apprendre à créer rapidement des interfaces modernes
Développer des compétences utiles en entreprise
Me préparer à mes projets

D) Environnement technologique

Visual Studio Code (Microsoft)
Navigateur web (Chrome, Firefox…)
Framework Bootstrap 5

E) Quelques notions à travers des images

Le système de grille (Grid System)

Bootstrap utilise un système de grille basé sur des lignes et des colonnes permettant de structurer facilement une page web et de la rendre responsive sur tous les écrans.

Les composants (Components)

Bootstrap propose de nombreux composants prêts à l'emploi comme les barres de navigation, boutons, cartes ou modales pour créer rapidement des interfaces modernes.

Les classes utilitaires

Les classes utilitaires permettent de modifier rapidement le style (marges, couleurs, alignements) sans écrire de CSS personnalisé.

Le responsive design

Bootstrap permet d'adapter automatiquement les pages web aux différentes tailles d'écran grâce à ses classes responsive (mobile, tablette, desktop).

Les formulaires Bootstrap

Bootstrap facilite la création de formulaires modernes et accessibles avec des styles intégrés et des mises en page optimisées.
